# 📊 Desafio 1: Análise de Desempenho - Alura Store

Este repositório contém a resolução de um desafio prático de Data Science, elaborado para o programa ONE em conjunto com a Alura, focado na análise exploratória de dados da rede fictícia **Alura Store**. 

## 🎯 Objetivo do Projeto
O Senhor João, proprietário da rede Alura Store, planeja iniciar um novo empreendimento. Para financiar essa nova etapa, ele precisa vender uma de suas quatro lojas atuais. O objetivo deste projeto é investigar os dados de vendas, métricas de desempenho e avaliações de clientes para identificar a unidade com a menor eficiência, entregando uma recomendação estratégica e baseada em dados sobre qual filial deve ser vendida.

## 🔍 Análises Realizadas
Para embasar a decisão de negócio, foram explorados e visualizados os seguintes indicadores de cada filial:
- **Faturamento total** por loja.
- **Categorias de produtos** mais e menos vendidas.
- **Média das avaliações** de satisfação dos clientes.
- **Volume de vendas**, destacando os produtos com maior e menor saída.
- **Custo médio do frete** e seu possível impacto nas vendas.

## 💻 Tecnologias e Ferramentas
- **Python** (Linguagem principal)
- **Pandas** (Limpeza e manipulação de dados)
- **Matplotlib** (Visualização de dados)
- **Jupyter Notebook / Google Colab** (Ambiente de desenvolvimento e documentação)

## 📁 Estrutura do Repositório
- `AluraStoreBrasil.ipynb`: Notebook principal contendo todo o código, a exploração dos dados, a geração de gráficos e o relatório final com a recomendação de negócios para o Senhor João.

## 📈 Resultados
A análise completa culminou em um relatório gerencial, apontando de forma objetiva qual loja apresenta os maiores gargalos financeiros (como o menor faturamento) e operacionais, garantindo que a tomada de decisão seja a mais segura e rentável possível para o futuro da marca.
